```text
│   ├── banner.svg              → animated hero banner (name, title, tagline)
│   ├── hero.svg                → animated neural-network illustration

│   ├── background.svg          → subtle grid + particle background
│   ├── avatar-frame.svg        → rotating gradient ring for the footer mark

└── CONTRIBUTING.md             → contribution guidelines for this repo
```

</details>

<details>
<summary><b>Design notes</b></summary>

- **Palette:** a consistent `#4facfe → #a06bff → #00f2fe` gradient family over a near-black `#05040f` / `#0a0e27` base, chosen for an AI / futuristic feel that still passes contrast checks against light text.
- **Animation strategy:** all motion in `assets/*.svg` uses native SVG `<animate>` / `<animateTransform>` / `<animateMotion>` rather than JavaScript, since GitHub's image proxy strips `<script>` tags but renders SMIL and CSS animation correctly inside `<img>` tags.
- **Stats widgets:** GitHub stats, streak, top languages, trophies, and the activity graph are rendered by third-party open-source services ([github-readme-stats](https://github.com/anuraghazra/github-readme-stats), [github-readme-streak-stats](https://github.com/DenverCoder1/github-readme-streak-stats), [github-profile-trophy](https://github.com/ryo-ma/github-profile-trophy), [github-readme-activity-graph](https://github.com/Ashutosh00710/github-readme-activity-graph)) — no API keys required.
- **Snake animation:** generated by [Platane/snk](https://github.com/Platane/snk) via `.github/workflows/snake.yml`, which pushes the rendered SVGs to an `output` branch on a daily cron schedule and on every push to `main`.
- **Accessibility:** every image/SVG tag includes descriptive `alt` text, and headings follow a single logical hierarchy so the table of contents anchors resolve correctly.

</details>
